# zkdj-gulp-cli
    为团队不人肉搬运gulp的代码，写了人生第一个脚手架。
    
    
###安装

    npm innstall -g zkdj-gulp-cli
    
###使用方法
 >1、查看模版列表
    
    $ zkdj list
    
    可用的模版列表：
    
      ★  mobile-gulp-template - 这是一个关于移动端适配方案的示例项目。
      ★  pc-gulp-template - PC端的一套模版

        
 >2、创建项目
 
    $ zkdj init
    ? 请选择模版 (Use arrow keys)
    ❯ mobile-gulp-template 
      pc-gulp-template 
      
    ? 请输入项目名字
